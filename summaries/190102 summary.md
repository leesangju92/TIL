## 1. Jupyter  사용법

* ```
  pip install jupyter
  cd TIL
  cd 01_python
  git clone 주소 붙여넣기
  cd jupyter_notebooks/
  ls -a
  rm -rf .git
  ls -a
  ```

* 실행하려면 git bash에서 `jupyter notebook`

* 초록색 = 에디트 모드, 파란색 = 커맨드 모드

* `ctrl + ENTER` 를 누르면 에디트 모드에서 실행된다.

* 셀 단위로 코드를 실행된다.

* 여러 단축기 : {단축키 소개:H,  아래 셀 추가:B, 에디트 모드로 전환:B, 지우기:D}
